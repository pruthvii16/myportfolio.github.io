Arduino Smart Dustbin Project

The Arduino Smart Dustbin is a clever project that uses a special sensor to open the dustbin lid automatically when you come near. This sensor, like magic, senses when you have some trash to throw away and opens the lid for you. It's like having a smart helper for keeping things clean. This project makes throwing away garbage easy and fun without even touching the dustbin lid!
